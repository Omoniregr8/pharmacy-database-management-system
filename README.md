# Pharmacy Database Management System

A real-world pharmacy management system built to help pharmacies manage:

- Medicines inventory
- Sales records
- Receipt generation
- Dashboard overview
- Data organization and tracking

## Features
- Add and manage medicines
- View sales history
- Generate receipts
- Simple dashboard
- Open access (no login)

## Purpose
This project was built to demonstrate how technology can improve
pharmacy operations, record keeping, and efficiency.

## Tech Stack
- Programming logic; Python
- Database management; for testing sqlite3 but for production PostgreSQL
- System design; Pure Html Css ie bootstrap and Js
